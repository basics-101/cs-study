## ✍️ 진행 방식
- 온라인 (월/수/금)
    1. 과목 학습(정의, 목적, 장/단점)
    2. 답변 작성 및 학습하며 나온 꼬리질문은 각자 알아서 조사(꼬리 질문은 깃헙 커밋X)
    3. 주제/주차/이름.md 형태로 작성
- 오프라인 (일)
    1. 랜덤으로 주제를 고르고 모의 면접 진행(각자 준비한 꼬리질문 기반) 및 피드백
    2. 꼬리질문 취합해서 주제/주차/README.md에 추가
    3. 다음주 질문 선정
